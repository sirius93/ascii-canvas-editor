# ASCII Canvas Editor

A powerful web-based ASCII diagram editor for creating beautiful ASCII art diagrams and drawings.

## Features

- **Multiple Drawing Tools**: Rectangle, Diamond, Line, Arrow, Text, Freehand, and Eraser tools
- **Border Styles**: Simple, Double, Rounded, Heavy, and Light border styles
- **Line Styles**: ASCII, Light, Heavy, Dashed, and Dotted line styles
- **Arrow Heads**: Standard and Triangle arrow head styles
- **Undo/Redo**: Full undo and redo functionality
- **Zoom & Pan**: Mouse wheel zoom and space+drag panning
- **Copy to Clipboard**: Export your ASCII art with one click
- **Keyboard Shortcuts**: Efficient keyboard shortcuts for all tools

## Usage

1. Open `index.html` in your web browser
2. Select a tool from the toolbar
3. Click and drag on the canvas to draw
4. Use keyboard shortcuts for quick tool switching
5. Copy your ASCII art to clipboard

## Keyboard Shortcuts

- **V**: Select/Move tool
- **R**: Rectangle tool
- **D**: Diamond tool
- **L**: Line tool
- **A**: Arrow tool
- **T**: Text tool
- **F**: Freehand tool
- **E**: Eraser tool
- **Ctrl+Z**: Undo
- **Ctrl+Y**: Redo
- **Ctrl+Shift+C**: Copy ASCII
- **Space+Drag**: Pan canvas
- **Scroll**: Zoom in/out
- **Delete/Backspace**: Delete selected shapes
